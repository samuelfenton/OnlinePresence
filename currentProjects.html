Ragtime Heist

Ragtime heist is a turn based tactical stealth game created which was crated over the course of 5 months where I was the solo programmer. Key components of this project include the development of GOAP (What is GOAP) to allow NPC's in the level to dynamically react to what happens. This was also my first attempt into using root motion for all movement of players.

Faber Nights

Faber Nights is a 2.5D side scrolling platformer, currently in the early stages of development. Its most prominent feature is the method of navigation. While traversing left and right the path itself bends around the world as seen below. This allows a 2D game to include more options for pathing, and overall changes how the environment is crafted.

Voxeliser

Voxeliser is my first attempt into custom shaders. This geometry shader is focused on converting an already existing model into a voxel counterpart. Also allowing for animations to be played in real time with voxel placement being locked into a world "grid".
