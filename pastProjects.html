Neon Odyssey

Neon Odyssey was my first major project being developed over 4 months. It is a co-op twin stick side scroller. Although it was nothing too spectacular; I was able to learn a lot about behavior trees for AI as well as movement driven by ray-casting rather than a rigid body.

Global Game Jam 2018 - Data Drone

For the 2018 Global Game Jam, teams required to create a game with the theme transmission (Data Drone on Global Game Jam). We decided to create Data Drone. Players would be able to transfer between different robot types in order to achieve the goal or reaching the end of the level. This created the challenge of implementing three different control schemes and designing puzzles in such a small time.

CMD Clicker

CMD Clicker is an implementation of a clicker game in a command line. It included ASCII art for a basic UI, and keyboard inputs. As an assignment piece this was my first step into customizing consoles.
