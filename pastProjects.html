Neon Odessy was my fist major project being developed over 4 months. It is a co-op twin stick side scroller. Although it was nothing too spectacular; I was able to learn a lot about behaviour trees for AI as well as moveent driven by raycasting rather than a rigidbody.

Global Game Jam 2018 - Data Drone

For the 2018 Global Game Jam, temas required to create a game with the theme transmission(Data Drone on Global Game Jam). We decided to create Data Drone. Players would be able to transfer between different robot types in order to achieve the goal or reaching the end of the level. This created the challange of implementing three different control schemes and designing puzzles.
